A long-form, self-contained HTML growth strategy document built for Foremind — a workplace mental health company — delivered as a living page instead of a PDF. Custom NBH-branded layout using Lay Grotesk, a soft pastel palette (peach, butter, mauve, sage, mint), and a single `index.html` with all styles inlined for portability.

> **Important** — This repo is archived. The Foremind Resource Hub build has since moved to HubSpot CMS plus HubDB (gated content, 200+ resources). Kept here as the reference deck that seeded that engagement.

## Usage

Open `index.html` directly in a browser, or serve the repo as a static site on any host (GitHub Pages, Netlify, or similar). No build step, no dependencies, no tracking.

```bash
# Local preview
open index.html

# Or serve as a static site
python3 -m http.server 8000
```
