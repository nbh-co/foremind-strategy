# Foremind Growth Acceleration Plan

Static HTML strategy deck built for Foremind — a workplace mental health company — delivered as a hosted alternative to a PDF proposal.

> **Client:** Foremind (foremind.com.au) | **Industry:** Mental Health Care | **Status:** Archived

## What It Is

A long-form, self-contained HTML growth strategy document delivered as a living page instead of a PDF. Custom NBH branded layout using Lay Grotesk, a soft pastel palette (peach / butter / mauve / sage / mint), and a single `index.html` with all styles inlined for portability.

## Structure

```
index.html              # Full strategy document
fonts/                  # Lay Grotesk trial OTFs (regular → black)
nbh-logo-black.png      # NBH branding assets
nbh-logo.svg
```

## Usage

Open `index.html` directly in a browser, or serve the repo as a static site (GitHub Pages / Netlify / any static host). No build step, no dependencies, no tracking.

## Status

Archived. The Foremind Resource Hub build has since moved to HubSpot CMS + HubDB (gated content, 200+ resources). This repo is kept as the reference deck that seeded that engagement.
